关于 GitHub热榜

这是一个自动获取并展示GitHub每周热榜项目的网页应用。

功能特性：

  • 📊 自动获取GitHub热榜项目数据
  • 🎨 现代化的卡片式UI设计
  • 🔍 支持按编程语言和关键词搜索
  • 📱 响应式设计，适配各种设备
  • ⚡ 通过GitHub Pages快速部署

技术栈：

  • 前端：HTML5 + CSS3 + JavaScript
  • 数据获取：Python + GitHub API
  • 自动化：GitHub Actions
  • 部署：GitHub Pages

数据每周一凌晨自动更新，确保您看到的是最新的热门项目。
